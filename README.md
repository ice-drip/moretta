# @kaffee/moretta

A enhance linter util

## Installation

Install with npm

```bash
  npm install --dev @kaffee/moretta
```

Install with yarn

```bash
  yarn add --dev @kaffee/moretta
```

Install with pnpm

```bash
  pnpm add -D @kaffee/moretta
```

## Usage/Examples

Example command

```bash
yarn moretta --file "src/**/*.{ts,html}"
npx moretta --file "src/**/*.{ts,html}"
pnpm moretta --file "src/**/*.{ts,html}"
```
